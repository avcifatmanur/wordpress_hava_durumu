Basic WordPress Weather Plugin .


== Installation ==

Eklentinin nasil kurulacagi:

1. Dosyayi `/wp-content/plugins/hava-durumu` dizinine ekleyebilir ya da WordPress eklentileri ekranindan dogrudan yukleyebilirsiniz.
2. Eklentiler ekranini kullanarak eklentiyi etkinlestirin.



== Description ==

Bu eklenti  hava durumunu dinamik sekilde sitenizi ziyaret edenlerin goruntulenmesini saglar.

Eklentiyi yukleyip etkinlestirdikten sonra:

*   Ilk olarak bir API key almaniz gerekiyor, http://openweathermap.org/appid adrese girdikten sonra buradaki “Sign up” linkinden kayit olusturacaksiniz. 
*   Kayit olusturunca hemen sonraki sayfada API Key bilgisi gelecek. Bu API Key bilgisini kaydedin.
*   Havadurumu bilgilerini sorgularken bu API key’i kullanacaksiniz.
*   Dashboard ->Gorunum ->Bilesenler -> SICAKLIK  yolunu izleyerek eklentiyi istediginiz alana ekledikten sonra API KEY bolumune aldiginiz apikeyi yapistirin.
